How to install ReNomQ
======================
ReNomQ can be downloaded from the URL below.

URL: https://github.com/ReNom-dev-team/ReNomQ.git ::

  git clone https://github.com/ReNom-dev-team/ReNomQ.git
  cd ReNomQ
  pip install -r requirements.txt
  pip install -e .

Requirements
^^^^^^^^^^^^^^
* Linux/Ubuntu
* Python 3.6
* numpy 1.15.4
* matplotlib 3.0.2
* ply 3.11
* networkx 2.2
* Pillow 6.0.0
* sympy 1.3
